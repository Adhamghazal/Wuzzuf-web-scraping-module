# Wuzzuf-web-scraping-module
## Steps to use:
1-First of all downlaod and import the module 
2-Call the scrap function and provide it with the field or your searching target 
3-Run the module and it will automatically scrap all of the pages related to the provided search target returning an dataframe which you can save it as excel file to use later with the follwoing features:
  -Job Title
  -Job Type
  -Location
  -Requirements
  -Job Link
